    • map()

      • 映射函数 先将数组中每一个元素遍历 元素经过指令变换 生成新的元素 返回新的元素后会将返回的所有元素形成一个新的数组

      • let name = ['hy','tt','zz']
let newname = name.map(function(t){
    return t + 'p'
})
console.log(newname)
